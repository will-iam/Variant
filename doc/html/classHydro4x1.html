Solves mass/transport equation on given subdomain.

This function computes one iteration in time of the transport equation on a given subdomain on shared memory. Solving on a "small" portion of the global domain such as a SDS leaves control to the user on how the computations are done (parallel threads, geometrical shape of the computed domain, etc.)
